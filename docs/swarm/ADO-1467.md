## Goal
Implementare `.github/workflows/skill-security-scan.yml` nel repo `xrmghost-skills` con upload SARIF nel Security tab e blocco dei finding critici.

## Scope e modifiche
- Creato `.github/workflows/skill-security-scan.yml`.

## Rischi residui
- Il fallback con Trivy soddisfa il vincolo di assenza di servizi esterni autenticati, ma non replica la semantica specifica di `cisco-ai-defense/skill-scanner` sugli skill prompt.
- La branch protection non è stata aggiornata: l'endpoint GitHub ha risposto `404`, probabilmente per permessi insufficienti o protezione non gestibile con il token corrente.
- Non è stato possibile eseguire una PR di test perché l'istruzione operativa di livello superiore vieta l'apertura di PR in questo run.
- `yamllint` e `actionlint` non sono disponibili localmente, quindi la validazione è stata limitata alla revisione statica del file generato.
